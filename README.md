# Игра «Угадай животное»
Проект по предмету "Основы программирования. Часть 2". Выполнен Токиной Екатериной и Михеевой Анастасией.
> ИТМО, ФТМИ, Бизнес-информатика. 1 курс

## Про игру
Для игры досточно запустить файл `main.py`.

Вам будут задаваться вопросы с ответом да/нет. Дойдя до конца ветки вопросов, Акинатор попробует угадать животное. Если он не справился, то запросит загаданное животное и вопрос, который отличает его от предположённого. Вопрос будет добавлен в виде новой ветви.

Помимо основной игры есть секция **статистики**. Там можно увидеть наиболее эффективные (т.е. чаще ведующие к победе) вопросы, структуру дерева, глубину раундов и общее число побед.

## О структуре проекта
- `main.py` — файл с интерфейсом/меню, объединяет все функции
- `guess_the_animal.py` — класс бинарного дерева со всеми ключевыми функициями, включая статистические
- `stack_question.py` — вспомогательный файл, где хранится класс вопросов

> Мы старались распределять работу как можно более равномерно: вместе продумали структуру, после чего **Анастасия** написала код ключевого функционала — дерева и стэка, а **Екатерина** его доработала, добавила функции для статистики и `main.py` *(ну и оформила этот репозиторий)*
